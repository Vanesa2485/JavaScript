El algoritmo creado para la primera pre-entrega es un silumador de una tienda de compras.
Descripción del algoritmo:
Al ajecutar el programa debe dar la bienvenida a la tienda, luego de eso, muestra un menu en prompt con dierentes opciones de grupos de artículos y en base a la opción elegida, utilizando alert muestra el grupo de artículos con sus precios. Si la entrada por prompt no corresponde a ninguna de las opciones del menu, debe dar un mensaje alertando "la opción seleccionada no es la correcta". Si el usuario elige 0 del menu, el programa debe terminar y luego da el mensaje de "Gracias por su visita".
